# Kursplan – Bootcamp (6 veckor)

## Översikt
Bootcampen är en introduktion till webbutveckling där studerande lär sig grunderna i HTML, CSS och JavaScript. Målet är att bygga en stabil grund innan den ordinarie kursen i Frontendutveckling startar.

Bootcampen är 6 veckor lång och består av tre huvudområden:

- HTML + CSS (2 veckor)
- JavaScript (3 veckor)
- Repetition + slutprojekt (1 vecka)

## Quiz varje vecka
I slutet av varje vecka görs ett kort quiz. Syftet är att studenten ska kunna se:
- vilken nivå man ligger på  
- hur långt man har kommit  
- vad man behöver träna mer på  
Genom quizzen kommer båda jag och studenten kunna se vilken nivå studenten ligger på, vad studenten behöver hjälp med och även enklare kontrollera att studenten klarar alla moment som ingår i kursen.

---

## Betyg: G och VG

### Godkänt (G)
- Studenten har genomfört och lämnat in alla moment och övningar som hör till varje vecka.  
- Studenten visar att hen förstår grunderna och kan använda dem i enklare uppgifter.

### Väl godkänt (VG)
- Studenten ska, utöver allt i G, genomföra ett sidoprojekt.  
- Sidoprojektet kan göras individuellt eller i par.  
- Projektet ska visa att studenten kan bygga en lösning från grunden, arbeta med tydlig struktur och använda kursens innehåll i praktiken.

## HTML-grunder

- Formulär och inputs  
- Grundläggande semantik  

## CSS-grunder
- Färger, text, spacing  
- Box-model  
- Klasser och id  
- Flexbox 
- Grid
- Enkel responsiv design  

## Mini-projekt
- Profilsida  
- Produktkort  
- Enkel layout med flexbox

# Vecka 3: JavaScript – Grunder

## JS Basics
- Variabler  
- Datatyper  
- If/else  
- Loopar  
- Funktioner  

## Struktur
- `script`-taggen  
- `console.log`

- Miniräknare i konsolen

# Vecka 4: DOM & Events

## DOM
- Hämta element (`querySelector`)  
- Ändra text, HTML och klasser  
- Skapa och ta bort element  

## Events
- `click`  
- `input`  
- `submit`  
- event handlers  

## Mini-projekt
- Räknare  
- Todo-lista  
- Modal / dropdown

# Vecka 5: JS + Struktur + Data

## Fördjupning
- Flera JS-filer  
- Export/import (basic eller simulerad)  
- Funktioner organiserade i mappar  
- Komponent-tänk  

## Datafil
- `mockdata` / `products.json`  
- Loopa igenom data  
- Skapa element från data  

## Mini-projekt
- Liten shop  
- Produktlista + filter  
- Sökfunktion  

---

# Vecka 6: Repetition & Slutprojekt

## Repetition
- HTML – snabb repetition  
- CSS – snabb repetition  
- JS-grunder  
- DOM + events  
- Datahantering  

## Slutprojekt
En liten webbapp som innehåller:

- HTML-struktur  
- CSS-design  
- JavaScript  
- DOM-hantering  
- Interaktion (knappar, inputs, events)  
- Data-fil (mockdata)  

### Exempel
- Mini-shop  
- Användarlista  
- Träningsapp  
- Dashboard med komponenter
